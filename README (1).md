## 🚀 Featured Projects

<details>
<summary><b>🏠 PropCheck AI — AI Property Due-Diligence Agent</b></summary>
<br/>

An AI agent that compresses a multi-hour manual property title-check into a 60-second due-diligence report, built for Indian lawyers.

| Category | Detail |
|---|---|
| **Stack** | Next.js, Python, Gemini 2.0 Flash, PostgreSQL |
| **Scale** | Covers property compliance rules across Maharashtra, Karnataka, Telangana & Uttar Pradesh |
| **Performance** | Reduces a multi-hour manual title-check to a ~60-second automated report |
| **Security** | Confidence-threshold validation layer routes uncertain fields to human review, not auto-approval |
| **Impact** | Correctly flagged owner-name mismatches, missing signatures & compliance gaps across 3 real sale deeds in testing |
| **Repository** | [GitHub](https://github.com/Himanshixyz2005) · Live Demo |

Implements a 5-stage pipeline — OCR, field extraction, normalisation, rule validation, and report generation — extracting 20+ structured fields from scanned, handwritten, and regional-language deeds using Gemini Vision. State-specific compliance rules are encoded as a validation layer. Validated through hands-on user testing with 8 law students and a practising legal professional, with their findings fed back into the pipeline.

</details>

<br/>

A voice-based consent platform replacing paper forms, explaining medical procedures to patients through real-time conversation in Hindi, Marathi, Bhojpuri, and English.

| Category | Detail |
|---|---|
| **Stack** | Next.js, Gemini, WebRTC, Supabase, C2PA |
| **Scale** | Real-time multilingual conversation across 4 languages |
| **Performance** | Live voice/video conversation over WebRTC with real-time AI response |
| **Security** | Tamper-evident C2PA consent receipts signed with voice & video, plus SHA-256 fingerprinting |
| **Impact** | Produces an auditable record of exactly what each patient was told before consenting |
| **Repository** | [GitHub](https://github.com/Himanshixyz2005) · Live Demo |

Replaces static paper consent forms with a live, conversational explanation of medical procedures. Every session produces a cryptographically signed, tamper-evident record — combining C2PA provenance standards with SHA-256 fingerprinting to make consent both accessible and verifiable.

</details>
