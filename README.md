<details>
<summary><b>🩺 ConsentLens — AI-Powered Multilingual Surgical Consent Platform</b></summary>
<br/>

An AI-driven platform that generates, translates, and verifies informed surgical consent in multiple languages, with real-time doctor-patient communication and tamper-proof consent verification.

| Category | Detail |
|---|---|
| **Stack** | Google Gemini 2.5 Flash, WebRTC, Supabase, Vercel, React |
| **Scale** | Multilingual pipeline supporting real-time translation & synthesis |
| **Performance** | Low-latency real-time AI response over WebRTC channels |
| **Security** | SHA-256 fingerprinting for consent integrity verification |
| **Impact** | Reduces informed-consent communication gaps in multilingual healthcare settings |
| **Repository** | *Private / available on request* |

ConsentLens tackles a critical healthcare communication gap — patients signing consent forms they don't fully understand. The system combines an LLM-driven translation layer with cryptographic fingerprinting so every consent instance is verifiable and auditable, while WebRTC enables live doctor-patient clarification sessions before consent is finalized.

</details>
