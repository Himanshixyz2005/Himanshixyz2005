## 🚀 Featured Projects

<details>
<summary><b>🏠 PropCheck AI — AI-Assisted Property Due-Diligence Agent</b></summary>
<br/>

A backend-driven agent that compresses a multi-hour manual property title-check into a 60-second due-diligence report, built for Indian lawyers.

| Category | Detail |
|---|---|
| **Stack** | Next.js, Python, Gemini 2.0 Flash, PostgreSQL |
| **Scale** | Property compliance rules across Maharashtra, Karnataka, Telangana & Uttar Pradesh |
| **Performance** | Multi-hour manual title-check reduced to a ~60-second automated report |
| **Security** | Confidence-threshold validation layer routes uncertain fields to human review, not auto-approval |
| **Impact** | Correctly flagged owner-name mismatches, missing signatures & compliance gaps across 3 real sale deeds in testing |
| **Repository** | [GitHub](https://github.com/Himanshixyz2005) · Live Demo |

Backend-first architecture: a Python-driven 5-stage pipeline — OCR, field extraction, normalisation, rule validation, report generation — extracts 20+ structured fields, with state-specific compliance rules encoded as a dedicated validation layer. Validated through hands-on testing with 8 law students and a practising legal professional.

</details>
